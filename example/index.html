Example Usage


<head>
  ...
  <!-- Import web component -->
  <script src="https://cbiit.github.io/nci-softwaresolutions-elements/components/include-html.js"></script>
</head>

<body>
  <header>
    <!-- Set component's src attribute to url of html snippet -->
    <include-html src="https://cbiit.github.io/nci-softwaresolutions-elements/banners/government-shutdown-test.html"></include-html>
    ...
  </header>
  ...
</body>

Rationale: HTML snippets (eg: official notices) can be managed centrally, and applications can include arbitrary markup without requiring a redeployment of application code. For example, the government shutdown banner html snippet can be updated/enabled/disabled within minutes, which will be reflected the next time the snippet is rendered by any application.

Security Concerns: Javascript is disabled within the snippet. However, loading any resources within the snippet incurs GET requests to the resource server, which can leak information. Always load resources from trusted sources.
