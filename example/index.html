Example Usage


<head>
  ...
  <!-- Import web component -->
  <script src="https://cbiit.github.io/nci-softwaresolutions-elements/components/include-html.js"></script>
</head>

<body>
  <header>
    <!-- Set component's src attribute to url of html snippet -->
    <include-html src="https://cbiit.github.io/nci-softwaresolutions-elements/banners/government-shutdown-test.html"></include-html>
    ...
  </header>
  ...
</body>

Rationale: The include-html web component enables centrally-managed HTML snippets (such as official notices) to be displayed in web applications without requiring a redeployment of application code. For example, the government shutdown banner html snippet can be updated/enabled/disabled within minutes, which will be reflected the next time the snippet is rendered by any application.


Enabling/Disabling: An optional comment can be included in the first line of the html snippet to enable or disable the snippet. Without this comment, the snippet is enabled by default.
To enable the snippet, use: <!-- ENABLED=TRUE -->
To disable the snippet, use: <!-- ENABLED=FALSE -->


Security Concerns: Javascript is disabled within the snippet. However, loading any resources within the snippet incurs GET requests to the resource server, which can reveal information about the client. Always load resources from trusted sources.
